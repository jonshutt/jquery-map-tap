Jquery UX Map.

A jQuery plugin designed to fix usability issues with Google Maps embedded on to web pages

Features:

  • Desktop users can use the mouse wheel to scroll down the page.
  • Mobile users can drag the map down without getting 'stuck' dragging the map.
  • Clicking, or touching the map 'enables' the map, allowing scroll wheel zoom.
  • Touching, dragging or scrolling away 'disables' the map.
  • Map height it limited so that mobile users don't 'stuck' dragging the map.
  • Map centers in the page so mobile users have an area above / below to drag away from the map.
  • Full screen view is especially useful for mobile users.

Simple Map

$( ".map" ).mapper();

Map, using callback to then add marker

$( ".map2" ).mapper({
    zoom:12,
    lat:55.9531,
    lng: -3.1889,
    complete:function(e) {			
                    
        var marker = new google.maps.Marker({
            position: {lat: 55.9531, lng: -3.1889},
            map: e.map
        });			
    }	
});
